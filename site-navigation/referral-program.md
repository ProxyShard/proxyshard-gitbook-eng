# Referral program

Every client can earn through the referral program.&#x20;

The "<mark style="color:purple;">Affiliate</mark>" tab contains a referral link. You can use it to invite friends and acquaintances to the website.

The referral reward system works as follows:

<figure><img src="../.gitbook/assets/image (253).png" alt=""><figcaption></figcaption></figure>

For every purchase made on the website by a referred person, you receive <mark style="color:purple;">**10%**</mark>.&#x20;

You can also specify a custom referral link. \
To do this, click "<mark style="color:purple;">Edit</mark>" next to the link and change it to your value.<br>

<figure><img src="../.gitbook/assets/image (254).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Information about link clicks and credited funds is not disclosed upon request to technical support
{% endhint %}
