# iOS / Android

## Programs for proxy setup on iOS / Android

### [<mark style="color:purple;">**V2Box**</mark>](v2box.md)

* Platform: <mark style="color:purple;">iOS</mark>, <mark style="color:purple;">Android, MacOS</mark>
* UDP support: Yes
* Difficulty: Medium
* Split routing: Added in recent updates

### [<mark style="color:purple;">**Potatso**</mark>](potatso.md)

* Platform: <mark style="color:purple;">iOS</mark>
* UDP support: Yes
* Difficulty: Easy
* Split routing: No, full tunneling

### [<mark style="color:purple;">**SuperProxy**</mark>](super-proxy.md)

* Platform: <mark style="color:purple;">Android</mark>
* UDP support: Yes
* Difficulty: Easy
* Split routing: No, full tunneling
