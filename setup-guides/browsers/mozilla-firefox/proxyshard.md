# ProxyShard Extension

Our official <mark style="color:purple;">**ProxyShard – Smart Proxy Manager**</mark> extension lets you manage proxies directly in Mozilla Firefox. Install it from **Firefox Add-ons**.

{% embed url="https://addons.mozilla.org/en-US/firefox/addon/proxyshard-smart-proxy-manager/" %}

{% hint style="success" %}
The Mozilla Firefox version supports <mark style="color:purple;">**SOCKS5**</mark> through the extension.
