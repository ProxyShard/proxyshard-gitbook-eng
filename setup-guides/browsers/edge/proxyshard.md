# ProxyShard Extension

Our official <mark style="color:purple;">**ProxyShard – Smart Proxy Manager**</mark> extension works in Microsoft Edge without restrictions, since Edge is built on the Chromium engine and supports extensions from the **Chrome Web Store**.

{% hint style="info" %}
When installing an extension from the Chrome Web Store for the first time, Edge may prompt you to «**Allow extensions from other stores**», please confirm this permission.
{% endhint %}
