## **Adding a connection profile**

Start the installed application, go to the bottom tab ![](<../../.gitbook/assets/image (91).png>), and add a new connection through ![](<../../.gitbook/assets/image (87).png>), specifying the <mark style="color:purple;">socks</mark> protocol.

## Proxy connection setup

Copy the connection from the order into the corresponding fields (IP/Hostname:PORT:USERNAME:PASSWORD) and click ![](<../../.gitbook/assets/image (90).png>). In "<mark style="color:purple;">Remark</mark>", you can specify any profile name.

<figure><img src="../../.gitbook/assets/Image00004 (1).PNG" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**You can find a proxy setup example in the [Setup guide](../getting-started.md) section**
{% endhint %}

## Connecting the profile and starting work

Select the required profile in ![](<../../.gitbook/assets/image (89).png>) if there are several, then go to ![](<../../.gitbook/assets/image (88).png>).

<figure><img src="../../.gitbook/assets/Image00005 (1).PNG" alt=""><figcaption></figcaption></figure>

Now you can click ![](<../../.gitbook/assets/image (92).png>) and the application will start working.

<figure><img src="../../.gitbook/assets/Image00003 (1).PNG" alt=""><figcaption></figcaption></figure>

**Done! You can now use the proxy.**
