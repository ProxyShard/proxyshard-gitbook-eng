## Installing ProxyCa&#x70;**.**

## **Routing rule setup**

For the proxy to work, adding a routing rule is required.

To add rules, open the "<mark style="color:purple;">Rules</mark>" tab and specify the settings as shown in the screenshot, making sure to set <mark style="color:purple;">Rule Action</mark>, <mark style="color:purple;">Proxy</mark>, and other parameters as needed.&#x20;

You can set separate proxying for processes by specifying the program path in the <mark style="color:purple;">Program</mark> field.

<figure><img src="../../.gitbook/assets/Image00003 (4).PNG" alt=""><figcaption></figcaption></figure>
